# agricultural-water-stress-mapping
this project assesses machine learning and remote sensing workflow for agricultural water stress prediction under climate and land use scenarios
## Objectives 
1.detect agricultural watewr stress
2. Analyze drought dynamics
3. develop predictive ML models
## data 
-MODIS
-Landsat
-sentinel-2
-rainfall
-temperature
## methods
-random forest 
-xgboost
-shap analysis
-NDVI/NDWI
## results 
the hybrid ensemble model combining XBoost and RF was applied to simulate agricultural water stress under multiple climatre and land use scenarios
1. model performance
overall accuracy :83.4%
kappa coefficient: 0.69
2. Climate water stress relationship: R^2=0.946
