# agricultural-water-stress-mapping

## Objectives 
1.detect agricultural watewr stress
2. Analyze drought dynamics
3. develop predictive ML models
## data 
-MODIS
-Landsat
-sentinel-2
-rainfall
-temperature
## methods
-random forest 
-xgboost
-shap analysis
-NDVI/NDWI
